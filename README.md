# DIVERSIFY HOME — Site Vitrine

## 🚀 Déployer sur Vercel en 3 étapes

### Étape 1 — Créer un compte Vercel
→ Aller sur **vercel.com** et créer un compte gratuit

### Étape 2 — Déployer le site
1. Cliquer **"Add New Project"**
2. Choisir **"Deploy from file upload"** ou connecter GitHub
3. Glisser-déposer le dossier `DIVERSIFY_HOME`
4. Cliquer **Deploy**

✅ Le site est en ligne en moins d'une minute.
